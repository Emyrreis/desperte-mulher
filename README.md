# Desperte Mulher - Flutter

Aplicativo desenvolvido em Flutter para realização de uma avaliação de risco da disciplina de Dispositivos Móveis,
O projeto tem como objetivo oferecer uma experiência simples, intuitiva e acolhedora
para o preenchimento de um questionário de avaliação.

## Objetivo

Adaptar a experiência do sistema web para dispositivos móveis, melhorando a
usabilidade, organização das informações e interação com o usuário.

## Tecnologias utilizadas

- Flutter
- Dart
